## Why need to freeze an object?
* be useful for representing a logically immutable data structure


## Use freezing

```
let object = {};
object.name = 'kasmine';
console.log(object); // { name: 'kasmine'}
// use freesing
let object = {};
Object.freeze(object);
object.name = 'kasmine';
console.log(object); // {}
Object.isFrozen(object); // true
```

Deep freezing is possible through Node.js packages such as [Deep Freeze](https://www.npmjs.com/package/deep-freeze).

```
let object = {
  name: 'kasmine',
  exp: {
    time: '2017-09-22'
  }
};
Object.freeze(object);
object.exp.time = 21;
console.log(object); // object: { exp: { time: 21 }, name: 'kasmine' };
```
