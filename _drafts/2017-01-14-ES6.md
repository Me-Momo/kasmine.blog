<small> 注: `...`扩展运算符是tage3的新内容，所以要安装 此依赖包</small>
```
npm install babel-preset-stage-3
```

###
```
"use strict"
var obj = {
  name: "A",
  age: "20"
}
var newObj = {
  name: "A",
  age: "23",
  birth: "1999年-08月"
};
obj = {...(obj), ...(newObj)
}
```
输出：
```
"use strict"
var obj = {
  name: "A",
  age: "20"
}
var newObj = {
  name: "A",
  age: "23",
  birth: "1999年-08月"
};
obj = {...(obj), ...(newObj)
}
```
