## 同时执行多个任务

`yield`指令可以很简单的将异步控制流以同步的写法表现出来

```javascript
// 错误写法，effects 将按照顺序执行
const users = yield call(fetch, '/users'),
      repos = yield call(fetch, '/repos')

const [uers, repos] = yield [
      call(fetch, '/users'),
      call(fetch, '/repos')
];
```
