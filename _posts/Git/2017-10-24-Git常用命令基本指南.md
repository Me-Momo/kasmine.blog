* git checkout
  * 从当前分支创建新的分支并且切换 `git checkout -b newbranch`

 
* git pull
  * git pull 操作就可以从远程库中获取某个分支的更新，再与本地指定的分支进行自动merge(即使本地不存在这个分支)
  * `git pull origin online:online`
 
* git branch
  * 查看本地所有分支 `git branch`

  * 查看远程所有分支 `git branch -r`
